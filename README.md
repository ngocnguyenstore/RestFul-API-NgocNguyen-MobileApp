# Tài liệu API Mobile App Ngoc Nguyen Store


## Thông tin Chung

- URL API: https://ngocnguyen.vn/api

- Cấu trúc Response

Nếu thành công:

```json
{
  "success": true,
  "data": [],
  "message": "success"
}
```

Nếu thất bại:

```json
{
  "success": false,
  "data": "",
  "message": "error message"
}
```
